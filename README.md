# platformer-engine-v3
A modular platformer engine inspired by Block Dash V2 (Khan Academy AI Contest). Built with p5.js and designed for easy block and player interaction customization.

## Features
- Scene management system
- Asset loader
- Player physics and stats
- JSON-based level design

## Getting Started
1. Change directories to Platformer
2. Run `python server.py`
3. Open `http://localhost:8000` in your browser
